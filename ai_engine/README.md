# DocuMind AI

**Local LLM-Powered Document Intelligence System (RAG)**

DocuMind AI is a privacy-first, offline AI system that lets users upload PDFs and ask natural-language questions. It uses Retrieval-Augmented Generation (RAG) with a locally hosted LLM to produce grounded answers with source context.
